# PedGraph -- Multidimensional network database for pedigree analysis

## Install

For now, we install via `pip`. Clone repo:

```
git clone https://github.com/olavurmortensen/pedgraph.git
cd pedgraph
```

Create a virtual environment:

```
python3 -m venv venv
```

Install via pip:

```
pip install -e .
```
